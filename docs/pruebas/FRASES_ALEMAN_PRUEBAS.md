# Frases en Alemán para Pruebas del Sistema

Este documento contiene frases en alemán con su traducción al español para realizar pruebas del sistema de reservas.

---

## Reservas - Creación

1. **Hallo, ich möchte einen Tisch reservieren.**  
   *Hola, me gustaría reservar una mesa.*

2. **Ich möchte für heute Abend einen Tisch für zwei Personen reservieren.**  
   *Quiero reservar una mesa para dos personas esta noche.*

3. **Können Sie mir einen Tisch für morgen um 20 Uhr reservieren?**  
   *¿Pueden reservarme una mesa para mañana a las 20:00?*

4. **Ich brauche eine Reservierung für vier Personen am Samstag.**  
   *Necesito una reserva para cuatro personas el sábado.*

5. **Haben Sie noch einen freien Tisch für heute um 19:30?**  
   *¿Tienen alguna mesa libre para hoy a las 19:30?*

---

## Reservas - Especificaciones

6. **Ich möchte einen Tisch am Fenster, bitte.**  
   *Quiero una mesa junto a la ventana, por favor.*

7. **Gibt es einen ruhigen Tisch für uns?**  
   *¿Hay una mesa tranquila para nosotros?*

8. **Können wir draußen sitzen?**  
   *¿Podemos sentarnos fuera?*

9. **Ich brauche einen Tisch für eine Geburtstagsfeier.**  
   *Necesito una mesa para una celebración de cumpleaños.*

10. **Ist es möglich, einen Tisch im Nichtraucherbereich zu bekommen?**  
    *¿Es posible conseguir una mesa en la zona de no fumadores?*

---

## Reservas - Modificación

11. **Ich möchte meine Reservierung ändern.**  
    *Quiero modificar mi reserva.*

12. **Können Sie die Zeit auf 21 Uhr ändern?**  
    *¿Pueden cambiar la hora a las 21:00?*

13. **Ich möchte die Anzahl der Personen auf sechs erhöhen.**  
    *Quiero aumentar el número de personas a seis.*

14. **Können wir die Reservierung auf nächste Woche verschieben?**  
    *¿Podemos posponer la reserva para la próxima semana?*

15. **Ich möchte den Namen der Reservierung ändern.**  
    *Quiero cambiar el nombre de la reserva.*

---

## Reservas - Cancelación

16. **Ich muss meine Reservierung stornieren.**  
    *Tengo que cancelar mi reserva.*

17. **Können Sie meine Reservierung für heute Abend absagen?**  
    *¿Pueden cancelar mi reserva para esta noche?*

18. **Ich möchte meine Buchung rückgängig machen.**  
    *Quiero anular mi reserva.*

19. **Leider kann ich heute nicht kommen, bitte stornieren Sie die Reservierung.**  
    *Desafortunadamente no puedo venir hoy, por favor cancelen la reserva.*

20. **Ich muss die Reservierung für morgen absagen.**  
    *Tengo que cancelar la reserva para mañana.*

    *¿Qué tienen hoy en el menú?*

22. **Können Sie mir die Speisekarte zeigen?**  
    *¿Pueden mostrarme el menú?*

23. **Was empfehlen Sie?**

24. **Gibt es vegetarische Optionen?**

25. **Was ist das Tagesgericht?**

## Pedidos - Especificaciones

    *Sin cebolla, por favor.*

    *¿Pueden hacerlo sin ajo?*

    *Soy alérgico a los frutos secos.*

## Consultas Generales

36. **Wie spät öffnen Sie?**  
    *¿A qué hora abren?*

37. **Bis wann sind Sie geöffnet?**  
    *¿Hasta qué hora están abiertos?*

38. **Haben Sie noch Plätze frei?**  
    *¿Tienen plazas libres?*

39. **Wie lange ist die Wartezeit?**  
    *¿Cuánto tiempo de espera hay?*

40. **Können Sie mir die Adresse geben?**  
    *¿Pueden darme la dirección?*

## Confirmaciones y Respuestas

41. **Ja, das passt mir gut.**  
    *Sí, eso me viene bien.*

42. **Perfekt, vielen Dank!**  
    *Perfecto, ¡muchas gracias!*

43. **Das klingt gut.**  
    *Eso suena bien.*

44. **Nein, das passt mir nicht.**  
    *No, eso no me viene bien.*

45. **Können Sie mir eine Bestätigung geben?**  
    *¿Pueden darme una confirmación?*

---

## Situaciones Especiales

46. **Ich habe eine Reservierung unter dem Namen Müller.**  
    *Tengo una reserva a nombre de Müller.*

47. **Können Sie mir helfen, eine Reservierung zu finden?**  
    *¿Pueden ayudarme a encontrar una reserva?*

48. **Ich bin zu spät, ist meine Reservierung noch gültig?**  
    *Llego tarde, ¿sigue siendo válida mi reserva?*

49. **Können wir den Tisch länger behalten?**  
    *¿Podemos mantener la mesa más tiempo?*

50. **Ich möchte eine Reservierung für eine große Gruppe machen.**  
    *Quiero hacer una reserva para un grupo grande.*

## Frases Coloquiales

58. **Hey, habt ihr noch was frei?**  
    *Oye, ¿tienen algo libre?*

59. **Könnt ihr mir was reservieren?**  
    *¿Pueden reservarme algo?*

60. **Das wäre super!**  
    *¡Eso sería genial!*

---

## Instrucciones para las Pruebas

1. **Usar estas frases en llamadas reales** para probar el sistema de reconocimiento de voz y procesamiento de lenguaje natural.

2. **Variar el tono y velocidad** al pronunciar las frases para probar la robustez del sistema.

3. **Probar frases mixtas** (alemán con palabras en otros idiomas) para verificar la detección de idioma.

4. **Probar con diferentes acentos** si es posible (alemán estándar, suizo, austriaco).

5. **Anotar los resultados** de cada prueba para identificar áreas de mejora.

---

**Nota:** Este documento está diseñado para facilitar las pruebas del sistema con hablantes nativos de alemán. Las traducciones al español están incluidas para referencia rápida.
